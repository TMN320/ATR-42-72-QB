# ATR Question Bank — Phone-first PWA

## Install on phone
1) Upload the files in this folder to any static hosting (GitHub Pages / Netlify / Vercel / any server).
2) Open the link on your phone:
   - Android (Chrome): menu → Add to Home screen
   - iPhone (Safari): Share → Add to Home Screen

## Quick local test
If you have Python:
- run in this folder:  python -m http.server 8000
- open: http://localhost:8000
